# Tp3_Symfony

## Description

Ce projet consiste à transformer une page HTML statique en une page dynamique avec Symfony.

## Travail réalisé
- Création d’une page produit avec Symfony
- Utilisation de Twig pour afficher les données du produit
- Intégration de Bootstrap pour le design
- Création d’un **Form Type Symfony** pour le formulaire (quantité et couleur)
- Traitement du formulaire au niveau du Controller
